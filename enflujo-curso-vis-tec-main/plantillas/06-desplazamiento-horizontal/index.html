Plantilla de scroll horizontal

Una historia que se lee como recorrido lateral

Usa esta estructura cuando el proyecto necesite comparar etapas, territorios, momentos o escenas en una secuencia horizontal.

Nombre Apellido - Fuente de datos - Fecha

El desplazamiento horizontal puede convertir una lista de hallazgos en un recorrido: cada panel funciona como una estación narrativa.

La persona hace scroll hacia abajo, pero la página mueve la historia hacia los lados. Este patrón sirve para líneas de tiempo, rutas, comparaciones por ciudad, procesos paso a paso o recorridos visuales.

Inicio

Presentar el punto de partida

Abre el recorrido con una pregunta clara. Antes de avanzar, explica qué se compara y por qué el orden de los paneles importa.

Estación 1

Primer cambio visible

Usa este panel para mostrar el primer giro de la historia: una subida, una concentración, una ausencia o una diferencia territorial.

Estación 2

Comparar lugares o grupos

El recorrido lateral permite sostener una misma pregunta mientras cambian los casos. Aquí puedes reemplazar la ilustración por un mapa o un gráfico.

Estación 3

Detenerse en un caso

Después de comparar, acerca la lectura. Un panel de detalle ayuda a explicar una excepción, una historia local o una consecuencia concreta.
